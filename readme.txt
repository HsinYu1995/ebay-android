The ebay app in android
The application was based on Pixel 2 XL test system (API 30 on android studio)

The image is based on AndroidImages/images 

The main xml file is located in hw9/app/src/main/res

The main java file is located in hw9/app/src/main/java/com/example/hw9/

The application request to the server set up on GCP which works as a proxy to send request to Ebay
